# Changelog

## [1.0.0] - 2026-03-16

### Added
- Landing page with repo URL input and demo buttons
- Route Tracer — horizontal swimlane visualization of API route execution chains
- Module Map — D3 force-directed graph of file dependencies with cluster detection
- Call Graph — dagre hierarchical function call visualization with dead code detection
- AST parsing via @babel/parser for JavaScript/TypeScript/JSX/TSX
- Route detection for Express, Next.js App Router, Next.js Pages API, Fastify, Hono
- Module dependency analysis with cyclomatic complexity calculation
- Function call tracing across files with layer classification
- AI annotations via OpenAI GPT-4o-mini (optional)
- Source code side panel with syntax highlighting via Shiki
- Search bar with autocomplete for files and functions
- Depth slider for controlling call graph exploration depth
- GitHub REST API integration for fetching public repo contents
- In-memory caching for analysis results
- Responsive layout for tablet and mobile
- GitHub Actions CI for build and lint
- Vercel deployment
